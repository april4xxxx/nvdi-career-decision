## 这一次，由女性执掌自己的时间王朝

面对模糊、复杂又充满情绪的职场困境，我们真正缺少的往往不是更多建议，而是看清：**为什么要做、需要投入多少、能够得到什么、机会成本是什么，以及此刻最值得专注的事情是什么。**

《女皇入朝》把每一个现实困惑变成一封等待御览的奏折。你不是等待安排的臣民，而是拥有最终决定权的女皇；AI 则化身辅政大臣，帮助你拨开信息迷雾，找到真正的决策点。

## 项目亮点

### 👑 决定权始终属于用户

AI 提供分析、追问和建议，却不会替你做决定。你可以通过“同意、再议、大胆”完成朱批，接受方案、继续追问，或挑战大臣给出的答案。

### 🏛️ 三位 AI 大臣，三种思考视角

直臣理性拆解问题，顺臣温和理解处境，卦师提供启发式推演。同一个困境可以切换不同文风重新审视，让 AI 的个性真正参与决策，而不只是为聊天框换上古风外壳。

### 📜 把完整决策过程变成一场宫廷冒险

现实困惑成为“奏折”，AI 分析成为“大臣议政”，确认后的行动进入朝堂、六部与御花园；完成记录写入起居注，阶段成果陈列于珍宝阁。

> **提出问题 → 看清取舍 → 确认行动 → 专注执行 → 沉淀经验**

### 🏆 奖励真实完成，而不是让人变得更忙

珍宝阁包含 **59 项独立设计的成就**。它们奖励每一次清醒决策、专注投入与真实行动，让游戏化成为成长的反馈，而不是另一份令人焦虑的任务清单。

### 🎨 一个细腻完整的女性帝王世界

人物、宫廷场景、奏折、任务卡与珍宝采用统一的东方美术体系。精美画面不仅提供沉浸感，也让那些艰难的职场决策拥有更从容、更有力量的表达方式。
